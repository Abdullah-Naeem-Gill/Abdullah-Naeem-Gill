### 2. **Human RL AI - Portfolio Ingestion Microservice**
> Production-grade microservice for automated broker portfolio ingestion from Alpaca Trading API with real-time tracking

**Project Duration:** 3 months

Built a robust, scalable microservice for automated ingestion of broker portfolio data from the Alpaca Trading API into PostgreSQL. Designed with clean async architecture and production-ready deployment on Google Cloud Platform, featuring OAuth 2.0 authentication, dual-database pipeline, and automated CI/CD workflows.

**Key Achievements & Responsibilities:**
- ‚úÖ **Microservice Architecture:** Built production-grade async FastAPI microservice with background task processing for real-time portfolio tracking
- ‚úÖ **API Integration:** Integrated Alpaca Trading API with robust error handling, retry logic, and connection pooling using SQLAlchemy ORM
- ‚úÖ **Authentication:** Implemented OAuth 2.0 token flow with automatic refresh handling and subscription-based access controls
- ‚úÖ **Database Design:** Created dual-database ingestion pipeline optimized for time-series storage and historical snapshot queries
- ‚úÖ **Service Architecture:** Built modular service layers for ingestion, account discovery, token management, and data enrichment
- ‚úÖ **Containerization:** Fully containerized the service with production-ready Docker configuration
- ‚úÖ **Cloud Deployment:** Deployed on Google Cloud Run with Cloud Scheduler for minute-level cron jobs and Artifact Registry integration
- ‚úÖ **CI/CD Pipeline:** Implemented automated CI/CD using GitHub Actions with Docker builds and environment-specific deployments
- üõ†Ô∏è **Tech Stack:** FastAPI | PostgreSQL | SQLAlchemy | OAuth 2.0 | Docker | Google Cloud Run | Cloud Scheduler | GitHub Actions | Alpaca API
